# Devenir Ingeson — Formations
Application de gestion des formations professionnelles (PWA).

---

## 🚀 Déploiement sur Vercel (10 minutes)

### Étape 1 — Créer un compte GitHub
→ https://github.com/signup (gratuit)

### Étape 2 — Créer un nouveau dépôt
1. Clique **"+"** en haut à droite → **"New repository"**
2. Nom : `devenir-ingeson-formations`
3. Visibilité : **Private** (recommandé)
4. Clique **"Create repository"**

### Étape 3 — Uploader les fichiers
1. Sur la page du dépôt, clique **"uploading an existing file"**
2. **Dézippe** l'archive et glisse **tout le contenu** dans GitHub
   ⚠️ Uploader le contenu du dossier, pas le dossier lui-même
3. Clique **"Commit changes"**

### Étape 4 — Déployer sur Vercel
1. → https://vercel.com → **"Sign up"** → **"Continue with GitHub"**
2. Clique **"Add New Project"**
3. Sélectionne `devenir-ingeson-formations`
4. Framework preset : **Create React App** (détecté automatiquement)
5. Clique **"Deploy"**

⏱ Attend 2-3 minutes...

### ✅ C'est en ligne !
Tu reçois une URL comme : `devenir-ingeson-formations.vercel.app`

---

## 📱 Installation sur téléphone (PWA)

### Sur iPhone (Safari uniquement)
1. Ouvre le lien dans **Safari**
2. Icône **Partager** (carré avec flèche ↑)
3. **"Sur l'écran d'accueil"**
4. **"Ajouter"**

### Sur Android (Chrome)
1. Ouvre le lien dans **Chrome**
2. Bannière **"Installer l'application"** → **"Installer"**
   Ou : menu ⋮ → **"Ajouter à l'écran d'accueil"**
